Forge PvP Sim

Monte-Carlo simulator for Forge Master arena fights — enter base values + % bonuses from the hero sheet, pick 3 skills, then Run.

Skill catalogue

Cooldowns are editable here (approximate — tune as you time them in-game; saved on this device) and hit counts are fixed — both apply to BOTH players. Damage is per player: set each hero's own dmg/hit next to the equipped slot in its panel (entered in millions; saved with presets). The APPLY_SKILL_* constants in the file control whether the global Skill Dmg/Hp/Cooldown % stats scale skills.
